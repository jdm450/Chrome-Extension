# Custom Chrome Theme Extension

## Getting the Code
To get a copy of this repository on your device, you can use Git to clone it. Open your terminal or command prompt and run:
git clone https://github.com/jdm450/Chrome-Extension.git

Alternatively, you can download the repository as a ZIP file directly from GitHub and extract it to a folder of your choice.

## Installing and Using the Extension in Chrome

Google Chrome allows you to load extensions and themes that are stored in a folder on your computer. This is known as loading an "unpacked" extension.

1. Access the Extensions Page

- Open your Google Chrome browser.
- navigate to chrome://extensions/
- Alternatively, click the three vertical dots in the top-right corner, select Extensions, and then click Manage Extensions.

2. Enable Developer Mode
- In the top-right corner of the Extensions page, locate the toggle switch for Developer mode.
- Ensure the toggle is switched to the On position.

3. Load the Unpacked Extension
- Once Developer mode is enabled, a new bar will appear with three buttons. Click Load unpacked.
- A file browser window will open. Select this folder titled, "Chrome-theme-extension-v1.0"

4. Create and Download Themes Using the Extension
- Open the theme generator dropdown by selecting the "Simple Theme Generator" Extension.
- Choose a color of your choice for the frame, toolbar, and background.
- Click "Confirm & Download JSON.
- Move this JSON file into an empty folder on your computer.

5. Load the Unpacked Theme Generated by the Extension
- Return to chrome://extensions and click Load unpacked.
- A file browser window will open. Navigate to the folder where you have stored your generated theme file (the folder containing the manifest.json file).
- Select the folder and click Open or Select Folder.
- Chrome will immediately apply the theme to your browser.
- If there are no errors in your manifest file, you will see the new theme reflected in your browser tabs and toolbar.
